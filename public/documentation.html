MyFlix API

This incredible API will let YOU look up movies!

MyFlix API is a server-side application that client-side developers can use to provide up-to-date, comprehensive movie info to users.

Description URL HTTP Method Request Response
Return a List of ALL movies to the user /movies GET None JSON: a list of all movies in the database with title, description, genre, director, image URL, whether it’s featured or not.
Example:

{
'Title': 'The Wizard of Oz',
'Description': 'Dorothy and Friends follow the Yellow Brick Road',
'Genre': 'Classic',
'Director': 'Victor Fleming',
'ImageURL': 'wizard_of_oz.png',
'Featured': 'Y'
}

Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user; use %20 for spaces /movies/:title GET None JSON: title, description, genre, director, image URL, whether it’s featured or not.
Example: See Above
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genres/:genre GET None JSON: description of a particular genre
Example:

{
'Genre': 'Comedy',
'Description': 'A genre that attempts to elicit laughter.'
}

Return data about a director (bio, birth year, death year) by name; use %20 for spaces /movies/directors/:name GET None JSON: Name, Bio, Birth Year, Death Year
Example:

{
'Name': 'Orson Welles',
'Bio': 'Orson Welles was an American actor, writer, and director who wrote and directed Citizen Kane.',
'Born': '1915',
'Died': '1985'
}

Allow new users to register /users POST JSON
Example:

{
'Username': 'johnny2345',
'Password': '12345',
'Email': 'johnny2345@example.com',
'Date of Birth': '01/01/2000'
}

Allow users to update their user info (username, password, email, date of birth) /users/:Username PUT See Above See Above
Allow users to add a movie to their list of favorites, use %20 for spaces /users/:Username/add/:Title POST None A text message confirming that the movie was added to favorites.
Allow users to remove a movie from their list of favorites /users/:Username/remove/:Title DELETE None A text message confirming that the movie was removed from favorites.
Allow existing users to deregister /users/delete/:Username DELETE None A text message confirming that the user was removed from the database.
